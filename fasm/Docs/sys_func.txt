ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ - ELF
ВХОДНАЯ ТОЧКА ФАЙЛА - main


                Функция 0 - вывести строку в консоль.

Выводит строку в консоль.
Параметры:
    eax = 0 - номер функции.
    ebx - строка которую функция должна вывести



                Функция 1 - получить кейкод с клавиатуры.

Получает номер клавиши после того как человек нажал на клавиатуру.
Номер клавишы сохраняется в регистр edx.
Параметры:
    eax = 1 - номер функции.
  

                Функция 2 - получить кейкод с клавиатуры.

Получает номер клавиши после того как человек нажал на клавиатуру.
Номер клавишы сохраняется в регистр edx и выводиться в консоль.
Параметры:
    eax = 2 - номер функции.

    
    
                Функция 4 - выделяет блок памяти.

Функция выделяет блок памяти, размером ebx байт, и возвращает указатель на начало блока.
Содержание выделенного блока памяти не инициализируется, оно остается с неопределенными значениями.
Результат сохраняется в edx.
Параметры:
    eax = 4 - номер функции.
    ebx - Размер выделяемого блока памяти в байтах.
  

                Функция 5 -  возвращает память назад в кучу.

Функция возвращает память, на которую указывает параметр ebx, назад в кучу.
Блок памяти, ранее выделенный с помощью вызова 4 функции освобождается.
Параметры:
    eax = 5 - номер функции.
    ebx - Указатель на блок памяти, ранее выделенный функцией 4.


                Функция 32 - рисует пиксель на экране.

Функция рисует пиксель с заданными параметрами на экране.
При успешной отрисовке возращает 1 в edx.
Параметры:
    eax = 32
    ebx - X координат пикселя.
    ecx - Y координат пикселя.
    edx - Цвет пикселя(в формате HEX, пример: "0x0000FF")

                Функция 33 - рисует линию на экране.

Функция рисует линию с заданными параметрами на экране.
При успешной отрисовке возращает 1 в edx.
Параметры:
    eax = 32
    ebx - X координат линии.
    ecx - Y координат линии.
    edx - X2 координат линии.
    esp - Y2 координат линии.
    ebp - Цвет пикселя(в формате HEX, пример: "0x0000FF")
